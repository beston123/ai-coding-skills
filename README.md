[![GitHub Stars](https://img.shields.io/github/stars/beston/ai-coding-skills?style=social)](https://github.com/beston/ai-coding-skills)

> A comprehensive collection of AI-powered coding skills and agents for enhanced code quality, efficiency, and reusability.

## ✨ Features

- **Code Quality Review**: Automated analysis for code quality, best practices, and maintainability
- **Code Reuse Analysis**: Identify opportunities for code modularization and DRY principle adherence
- **Efficiency Optimization**: Performance analysis and optimization recommendations
- **Code Refinement Tools**: Automated code improvement and refactoring capabilities

## 📦 Installation

Any AI coding tool that supports skills and agents can use this. Clone the repo and copy the files to your tool's configuration directories.

```bash
git clone https://github.com/beston123/awesome-skills.git
```

Copy `skills/<skill_name>` to your tool's skills directory, and copy all files under `agents/` to your tool's agents directory. Refer to your tool's documentation for the exact paths.

## 🚀 Usage

Invoke it in any session:

```
/code-refine
```
